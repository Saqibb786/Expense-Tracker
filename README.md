# Personal Expense Tracker

A simple command-line tool to track your personal expenses. This application allows you to manage your finances by adding expenses, tracking your balance, and viewing spending summaries.

## Features

- **Add Expenses:** Record expenses with a description, amount, and category.
- **Manage Balance:** Add funds to your balance to keep it updated.
- **View Balance:** Check your current balance at any time.
- **Expense History:** View a complete list of all your recorded expenses.
- **Categorical Spending:** See a breakdown of your spending by category.
- **Monthly Summary:** Get a summary of your expenses for each month.
- **Data Persistence:** All your data is saved locally in `.csv` files (`data.csv` for expenses and `balance.csv` for the balance).

## Getting Started

### Prerequisites

- Python 3.x

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/saqibb786/expense-tracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd expense-tracker
   ```

### Usage

To run the application, execute the `main.py` script from your terminal:

```bash
python main.py
```

The application will greet you and prompt you for a username. It will then present you with a menu of options to manage your expenses.
